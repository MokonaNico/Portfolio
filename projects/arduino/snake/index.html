Presentation of the project

This is one of my first project on arduino, not exactly the first but all the other project where just testing component and understanding how they work. It's an implementation of the game of snake with arduino on a 8x8 led matrix and a joystick for the control.

Implementation

Implementing the snake is easy, the only "hard" part is to understand how to make the part of the snake to follow the head, but when you get it it's easy to implement. For this project i use the 8x8 matrix led of elegoo to display the board of the game. The snake body is all the read dots and the apple is a red dot blinking. We can't tell the difference between the body and the head of the snake which is a bit bad but I can't do better with a unicolor led matrix.

To get the input of the player I use a joystick, only certain angle are used to detect a movement to a direction. Because the matrix don't make always clear of the situation of the game, I used a lcd screen to print some informations like the current lenght of the snake and the state of the game.
