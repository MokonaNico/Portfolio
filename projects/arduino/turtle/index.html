Presentation of the project

I did this project with my girlfriend when we just got the tft led screen of elegoo. We wanted to draw thing on the screen so the first idea that I got was to recreate something like the turtle in Python. So I made a turtle like the one in Python and we tried to draw different shape with this little turtle.

You can see different picture that we take of what we did with the turtle and the source code plus some explaination are in the implementaiton section.

Implementation

The first thing that I did was to create the Turtle object, it handle all the drawing stuff and it handle all the movement of the turtle like moving forward/backward, turning, ... With a turtle like this one, you can easily draw shape like polygon and do fractal figure with the recursion. You can see how me and my girlfriend did this in the source code.

To do this project, you need an elegoo tft lcd screen and an arduino. You can get the library to communicate with the screen on those links : Elegoo_TFTLCD & Elegoo_GFX. And the source code of the project is here : github link
