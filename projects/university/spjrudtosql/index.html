Presentation of the project

This is a little project that I had to do for the lesson of database. The objective of the project is to do an interpreter to translate SPJRUD into SQL. SPJRUD come from the relational algebra, S is for Selection, P for Projection, J is for Join, R is Rename, U is Union and D is Division. After the translation, the SQL request have to be done on a database. It was a project by group of 2.

I can't share the source code because it's a university project.

Implementation

We had to implement it in Python, to do it we created an object for each operation in SPJRUD and the parameters of the constructor of each operation were the parameters of the operation. For each object, we look for the best translation in SQL and implement it. At the end, we just add the database, it was a simple SQLite db.
