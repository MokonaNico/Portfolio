Presentation of the project

In this project, I created a simulation of "The Magnetic Pendulum". It's a simple experimentation with a magnetic pendulum under the influence of three magnets. In this project, you'll see how it shows the relationship between chaos, fractal and physics.

Images

On these images, you can how the pendulum move in a 2D space with the three magnets (In fact, I considered that the pendulum is above the magnet to have no issues when magnet and pendulum are overlapping). You can see that when we have near start position, the ending result is very different, that's where the chaos is.

Now we are going to see where the fractals are. For each pixel on the scene, we colored the pixel with the color where the pendulum stop. It takes some times to do all the work but when it's finish, we get the beautiful result that you can see just below.

Let's tweak some settings. We can change multiple value, like the friction, the pendulum pullback force and the magnet force. Here I'm going to modify only the first one. I show the result of modifying the friction just below. As you can see, when the friction is lower, it's like we are zooming out the fractal and more details/mayhem appear.

Implementation

I did this project in java with javafx for the graphics, I use this because that's what I learn at university and it's simple. My objective was to understand how to create a simulation that implies multiple formula of physics.
