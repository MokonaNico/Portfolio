Presentation of the project

The Julia Set is a set of complex number that we can find by iterating a function over all pixels of the screen. The iteration is z_new = z_old^2 + c where z and c are complex numbers. To be in the Julia Set, z must be in stay in the circle around the origin of radius 2.

The default value for z will be the coordinate on the screen and c is a number that we can choose. Now we can iterate until the number leave the circle, and to don't loop infinitely, we set a max iteration number. We do that for each pixels and we choose the coloration report to the number of iteration.

Implementation

Like my other project, I did this with Java and JavaFX. The first thing that I did was to create a complex number object to handle all operations about it. I make a canvas with JavaFX to drawn on each pixel and then I just calculate for each pixel the number of iteration. The last step is to choose the color, for this I create a color palette and I do the modulo of the number of iteration by the number of different color in my palette.
